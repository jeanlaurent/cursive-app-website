# Cursive Serpodile WebSite

should be up at http://jeanlaurent.github.io/cusrive-app-website

## pre-requisite (grunt)
```
npm install grunt-cli
```

## Install dep once
```
npm install
````

## Run in dev mode with livereload
```
grunt
```

## build locally
```
grunt build
```

## Deploy to gh-pages
```
grunt deploy
````
