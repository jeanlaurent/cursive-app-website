# Install and Run

## pre-requisite (grunt)
```
npm install grunt-cli
```

## Install dep once
```
npm install
````

## Run in dev mode with livereload
```
grunt
```

## Build locally
```
grunt build
```

## Deploy to gh-pages
```
grunt deploy
````
